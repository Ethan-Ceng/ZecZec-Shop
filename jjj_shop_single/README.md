ThinkPHP 6.0
===============

> 執行環境要求PHP7.1+。

## 主要新特性

* 採用`PHP7`強型別（嚴格模式）
* 支援更多的`PSR`規範
* 原生多應用支援
* 更強大和易用的查詢
* 全新的事件系統
* 模型事件和資料庫事件統一納入事件系統
* 模板引擎分離出核心
* 內部功能中介軟體化
* SESSION/Cookie機制改進
* 對Swoole以及協程支援改進
* 對IDE更加友好
* 統一和精簡大量用法

## 安裝

~~~
composer create-project topthink/think tp 6.0.*-dev
~~~

如果需要更新框架使用
~~~
composer update topthink/framework
~~~
